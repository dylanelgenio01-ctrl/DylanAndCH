Ini adalah Pembahasan kami tentang extension.



Domain Generik (gTLD – Generic Top Level Domain) Digunakan secara global tanpa batasan wilayah.

1. .com → untuk keperluan bisnis atau komersial.

2. .org → untuk organisasi nirlaba, sosial, atau komunitas.

3. .net → awalnya untuk jaringan atau ISP, kini digunakan umum.

4. .info → untuk situs yang menyediakan informasi.

5. .biz → untuk kegiatan bisnis atau usaha kecil.

6. .name → untuk situs pribadi.



Domain Negara (ccTLD – Country Code Top Level Domain) Menunjukkan asal atau lokasi geografis suatu situs.

1. .id → Indonesia.

2. .us → Amerika Serikat.

3. .uk → Inggris.

4. .jp → Jepang.

5. .au → Australia.

6. .sg → Singapura.
